SYNOPSIS: This is the CS317 project, running on Java code. This creates a non-determinisitic finite automata based off of the regular expression given from the user.
HOW-TO: To compile, type into the command-line: 'javac generateNFA.java regexToNFA.java' without the single quotes.

generateNFA.java - has code that creates NFAs using stacks.
regexToNFA.java - a program that accepts input and will generate a NFA based on the expression.
